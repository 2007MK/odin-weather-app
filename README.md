# 🌤️ Odin Weather App

**A sleek and functional weather dashboard built with Vanilla JavaScript. This application demonstrates the power of Asynchronous JavaScript by fetching and processing real-time meteorological data from an external API.**

### Project Context
This project was developed as part of **The Odin Project's** JavaScript curriculum. The primary objective was to master `async/await` syntax and the `Fetch API`, ensuring smooth data transitions and error handling when interacting with the **Visual Crossing Weather API**.

## Features
* **Real-time Data Fetching:** Fetches current weather and forecasts based on user-provided locations.
* **Dynamic Backgrounds:** UI elements change based on the weather conditions (e.g., sunny, rainy, cloudy).
* **Asynchronous UX:** Clean loading states and error handling for invalid location searches.
* **Modular Design:** Separation of API logic, data processing, and DOM manipulation.

## 🛠️ Tech Stack
* **Language:** Vanilla JavaScript (ES6+)
* **API:** [Visual Crossing Weather API](https://www.visualcrossing.com/weather-api)
* **Build Tool:** Webpack
* **Tooling:** ESLint, Prettier
* **Styling:** CSS3 (Flexbox/Grid)
